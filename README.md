[js-selection](http://aureooms.github.io/js-selection)
==

Selection algorithms for JavaScript. Parent is
[aureooms/js-sort](https://github.com/aureooms/js-sort).

```js
let quickselect = selection.singletco( partition.hoare ) ;
```
